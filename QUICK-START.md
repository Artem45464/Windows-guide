https://www.microsoft.com/software-download/windows11
- Select **Windows 11 (x64)** for Intel Macs

**Windows 10** (For older 2011-2015 Macs)
https://www.microsoft.com/software-download/windows10
- Select **Windows 10 (x64)** for Intel Macs

Not sure? Click **Apple Menu ‚Üí About This Mac** and look for "Processor: Intel"

## üöÄ Installation (5 mins)

### 1. Open Boot Camp Assistant
**Finder ‚Üí Applications ‚Üí Utilities ‚Üí Boot Camp Assistant**

### 2. Partition Your Drive
- Click **Continue**
- Select your **Windows ISO file**
- Choose partition size (default 64 GB is fine)
- Click **Install**
- ‚è≥ Wait for restart

### 3. Install Windows
- Boot Camp boots automatically
- Click **Install Now**
- Accept license terms
- Select **BOOT CAMP** partition
- Choose **Custom installation**
- ‚è≥ Wait 20-30 minutes

### 4. Activate Windows (Both 10 & 11)

After Windows boots:
- Press **Windows Key + I** (opens Settings)
- **Windows 11:** Go to **System ‚Üí About**
- **Windows 10:** Go to **Update & Security ‚Üí Activation**
- Click **Change product key** or **Activate**
- Enter your 25-character key
- Windows will activate within 1-24 hours ‚úÖ

(Without activation: watermark appears but Windows still works)

### 5. Install Drivers
- Open **File Explorer**
- Look for **Boot Camp** folder
- Run **BootCampSetup.exe**
- Restart when done

## ‚ö†Ô∏è If Something Goes Wrong

**Windows won't install?**
- Restart and try again
- Make sure you have 64+ GB free
- Disable FileVault in macOS first

**No trackpad/audio?**
- Install Boot Camp drivers (Step 5)
- Restart your Mac

**Windows won't activate?**
- Check your product key is correct
- Wait 24 hours (Microsoft validates in background)
- Try again if it still fails
